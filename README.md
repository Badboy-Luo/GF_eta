# GF_eta

## Introduction
Our algorithm shows how to calculate changes in geometric phase from Green's function.


$C_t (\omega) = \frac{1}{\sqrt{\sum_{n=1}^{N} \textbf{Re}(n)^2 + \textbf{Im}(n)^2} }$ 
	\left(
		\begin{array}{c}
			C^{1}e^{i\phi_{1}}	\\
			C^{2}e^{i\phi_{2}} \\
			... \\
			C^{N}e^{i\phi_{n}}		
		\end{array}
	\right)$
